# ProjetoDOW
Projeto de tecnologia assistiva para DOW Química.

Leitor de tabelas e criação de gráficos voltado a pessoas com limitações visuais.

INSTALL

Para rodar o programa, é necessário:

 - Python 3;

Dependências:

 - pandas;
 - simpleaudio;
 - matplotlib;
 - tkinter;
 - tkintertable;
 - xlsxwriter;

Por fim, executar o arquivo Tela1.py.
